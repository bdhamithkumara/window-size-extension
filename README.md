# Window Size Extension

**Window Size** is a lightweight browser extension that displays the current width and height of your browser window.

## Features
- Quickly view the exact dimensions (width and height) of your active browser window.
- Simple and easy-to-use popup interface.
- Works across all webpages.

## Installation

### For Google Chrome, Microsoft Edge, and Brave:
1. Download or clone this repository to your local machine.
2. Open your browser and navigate to the extensions page:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Brave: `brave://extensions/`
3. Enable **Developer mode** (usually a toggle switch in the top right corner).
4. Click on the **"Load unpacked"** button.
5. Select the `myextentionwhfinal` directory (the folder containing the `manifest.json` file).
6. The extension is now installed and ready to use! Click the extension icon in your browser toolbar to see the current window size.
